---

## 🔴 긴급 (Critical)

### 1. HTTPS 미설정
- [ ] 현재 HTTP만 사용 중
- [ ] 모든 데이터가 평문 전송
- [ ] 중간자 공격(MITM) 취약
- **해결:** Let's Encrypt 인증서 설치

### 2. AWS 액세스 키 노출
- [ ] .env 파일에 액세스 키 저장
- [ ] Git에 업로드될 위험
- [ ] 키 유출 시 AWS 계정 위험
- **해결:** IAM 역할로 전환

### 3. CORS 설정 부족
- [ ] 모든 origin 허용 중 (*)
- [ ] XSS 공격 가능
- **해결:** 특정 도메인만 허용

---

## 🟡 중요 (High)

### 4. JWT 인증 없음
- [ ] 전화번호만으로 인증
- [ ] 세션 탈취 가능
- **해결:** JWT 토큰 시스템

### 5. Rate Limiting 없음
- [ ] API 무제한 호출 가능
- [ ] DDoS 공격 취약
- **해결:** Express Rate Limit

### 6. 입력 검증 부족
- [ ] 사용자 입력 검증 부족
- [ ] SQL Injection, XSS 가능
- **해결:** 입력 검증 강화

---

## 🟢 권장 (Medium)

### 7. 프로세스 관리 없음
- [ ] nohup으로 수동 실행
- [ ] 자동 재시작 없음
- **해결:** PM2 설치

### 8. 모니터링 부족
- [ ] 에러 로그만 확인 가능
- [ ] 성능 메트릭 없음
- **해결:** CloudWatch 연동

### 9. 백업 전략 부족
- [ ] 수동 백업만
- [ ] 자동화 없음
- **해결:** DynamoDB 백업 활성화
